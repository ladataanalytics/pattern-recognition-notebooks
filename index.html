🐙 How to access the notebooks — GitHub + Google Colab setup
1
Create a GitHub account (free) at github.com/join if you don't have one. GitHub stores your notebooks and tracks your work across devices.
2
Fork the course repo. Go to the repo link in the header and click Fork (top-right). This creates your own private copy of all 29 notebooks under your GitHub account.
3
Open any notebook in Colab. Click the orange Open in Colab button on any card below. Colab is Google's free Python environment — no install needed, free GPU included.
4
Work through the notebook. Run cells top-to-bottom with Shift+Enter. Fill in # YOUR CODE HERE sections and complete the quiz at the bottom.
5
Save your work back to GitHub. In Colab: File → Save a copy in GitHub. Choose your forked repo. Your work is now versioned and accessible anywhere.
6
Track your progress here. Click any technique card → mark as complete or take the quiz. Progress is saved in this browser. Come back any time — no login required for this page.
Need help? First-time Colab users: go to colab.research.google.com and sign in with your Google account. First-time GitHub users: the GitHub fork guide takes about 5 minutes. You need a Google account for Colab and a GitHub account for saving — both are free.

🔧 One-time setup — takes about 5 minutes
Student quiz submissions flow into a Google Sheet you own. Here's how to set it up:
Step 1 — Create Google Form
Go to forms.google.com → Blank form.
Add these fields exactly:
1. GitHub Username (Short answer)
2. Notebook (Short answer)
3. Score (Short answer)
4. Total Questions (Short answer)
5. Answers JSON (Paragraph)
Step 2 — Link to Google Sheet
In the Form editor → Responses tab → click the Sheets icon (Link to Sheets) → Create new spreadsheet.

In the Sheet → Share → Anyone with the link can view.

Copy the Sheet URL and paste it below.
Step 3 — Get Form Entry IDs
Open your form → click ⋮ → Get pre-filled link.
Fill in dummy values, click Get link.
The URL will look like:
…?entry.123456789=username&entry.987654321=notebook…

Copy the entry.XXXXXXX IDs for each field and paste below.
Step 4 — Share with students
Once configured, the quiz cells in every notebook will show a "Submit to instructor" button.

Students click it → Google Form opens pre-filled → one click to submit.

You see results here in real time.
