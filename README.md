# 🔍 Pattern Recognition for the Rest of Us

A technique-based, non-linear machine learning learning hub. Jump to any topic — each notebook is self-contained with concept cells, runnable code, exercises, and a quiz.

**[→ Open the interactive navigator](https://ladataanalytics.github.io/pattern-recognition-notebooks)**

---

## How to use this repo

1. **Fork this repo** → you get your own copy to work in
2. **Click any "Open in Colab" badge** → notebook opens with free GPU/TPU
3. **Work through the notebook** → concept → code → exercises → quiz
4. **Commit your work back** → File → Save a copy in GitHub (from Colab)
5. **Track progress** on the [navigator page](https://ladataanalytics.github.io/pattern-recognition-notebooks)

## Notebook structure

Every notebook follows the same format so you always know where to look:

```
1. 🎯 What is this? (concept + intuition, no code)
2. 🧮 The math (plain notation, step by step)
3. 🐍 Python implementation (from scratch + sklearn)
4. 📊 Visualizations (plots that build intuition)
5. 💼 Real dataset exercise (you fill in the blanks)
6. 🧪 Quiz (5 multiple-choice, self-grading)
7. 📚 Further reading + cross-links to related techniques
```
